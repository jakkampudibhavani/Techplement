# Samsung Galaxy S24 Ultra Product Landing Page

A responsive static product landing page built using **HTML**, **CSS**, and **JavaScript**. The page showcases the Samsung Galaxy S24 Ultra with a product image slider, pricing section, product details table, description section, image gallery, navigation header, and footer links.

## Project Overview

This project is a front-end product landing page designed to display a featured smartphone product in a clean and simple layout. It focuses on presenting product information visually using responsive design, basic DOM manipulation, and CSS styling.

The page includes a sticky navigation bar, automatic product image slideshow, product pricing offer, specifications table, product description, gallery images, and footer sections such as company details, help links, products, and customer care information.

## Features

- Responsive product landing page layout
- Sticky header with navigation links
- Mobile-friendly hamburger menu
- Automatic image slider using JavaScript
- Product discount and pricing display
- Product details/specifications table
- About-this-item description section
- Product image gallery
- Footer with company, support, product, and contact links
- Custom styling with Google Fonts and Material Symbols

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Fonts
- Google Material Symbols

## How to Run the Project

1. Clone the repository:

```bash
git clone <your-repository-url>
```

2. Open the project folder:

```bash
cd Techplement-main
```

3. Open `index.html` in your browser.

No backend server or installation is required because this is a static front-end project.

## Important Note About Images

In `index.html`, the image paths are written like this:

```html
<img src="img/S24 ultra(1).webp" alt="Image 1" />
```
## JavaScript Functionality

The project uses JavaScript to control the product image slider. It hides all images first, displays the first image, and then automatically changes the image every 2 seconds.

It also includes a mobile menu toggle. When the menu icon is clicked on smaller screens, the navigation section expands or collapses.
